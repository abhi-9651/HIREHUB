# HireHub

## Project Theme

Employment & Recruitment Platform

## Project Overview

HireHub is a modern job portal that connects job seekers with recruiters. The platform allows users to browse job opportunities, create accounts, manage profiles, and apply for jobs through a simple and user-friendly interface.

## Features

* User Authentication (Login & Signup)
* Job Listings Dashboard
* Search and Browse Jobs
* Responsive UI Design
* User Profile Management
* Recruiter Job Posting (Planned)
* Application Tracking (Planned)

## Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS
* React Router DOM
* React Hook Form

### Backend (Planned)

* Node.js
* Express.js
* MongoDB
