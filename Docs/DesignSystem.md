# HireHub Design System

## Project Overview

**Project Name:** HireHub

**Tagline:** AI-Powered Career Copilot for Students

**Design Goal:** Create a modern, premium SaaS experience that feels like an AI startup rather than a traditional internship portal.

---

# Design Principles

### 1. Simplicity First

* Clean layouts
* Minimal clutter
* Clear visual hierarchy

### 2. Professional & Modern

* Startup-style UI
* Consistent spacing
* Smooth interactions

### 3. AI-Centric Experience

* Highlight AI features prominently
* Focus on Resume Studio and Career Copilot

### 4. Desktop First

* Primary design target: 1440px width
* Secondary support: tablets and mobile

Use startup-style naming.

Good:

* AI Resume Studio
* AI Career Copilot
* Smart Match
* Application Tracker

* Resume Generator
* Chatbot
* User Dashboard
* Internship Management
