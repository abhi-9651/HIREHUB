# HireHub Wireframes

## Project Overview

HireHub is an AI-powered career platform for students that helps users discover internships, build resumes, and receive AI-driven career guidance.

Target Device:

* Desktop First

Design Style:

* Modern SaaS
* Dark Theme
* Purple Accent
* Glassmorphism
* AI Startup Aesthetic

# Design Consistency Rules

* Use Dark Theme
* Use Purple Primary Color
* Use Cyan Accent Color
* Maintain Consistent Spacing
* Reuse Components
* Use Framer Motion for Animations
* Follow Desktop-First Layouts
* Keep UI Minimal and Professional

End Goal:

The product should look like a real AI startup platform rather than a college project.
